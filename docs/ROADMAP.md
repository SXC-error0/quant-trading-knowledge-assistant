# Roadmap

## V0.1：RAG MVP（当前工程）

- FastAPI 服务骨架。
- JSONL 知识块导入。
- Qdrant 向量检索。
- OpenAI-Compatible 模型接入。
- 来源返回与交易回答边界。

## V0.2：检索质量增强

- 增加全文检索 / BM25。
- 接入 Reranker。
- 增加问题类型路由与查询改写。
- 建立 100 条专业评测集和自动评测脚本。

## V0.3：产品集成

- 接入看盘前端。
- 增加用户反馈与人工修正记录。
- 接入“系统提醒详情”查询，仅解释已存在信号。
- 增加审计日志与访问控制。

## V0.4：行为微调评估

在真实反馈充分后，将人工确认的高质量答案整理为 ShareGPT 数据，仅针对回答格式、术语一致性和边界遵从进行 LoRA / QLoRA 实验。知识内容仍以 RAG 为主。
